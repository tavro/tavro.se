using ⚒️

this page was inspired by Stefan VanBuren and Shreyans Devendra Doshi, so thanks to them! here, i'll list the tools i use on a daily basis.

software

category home work comment
os linux (desktop & laptop), android (phone) linux (desktop), macOS (laptop) i always run the latest version of arch on my desktop linux machines, and i use the latest kali release on my personal laptop.
editor jakel android studio, xcode for private use, i'm using my own custom text editor. prior to implementing it, i was using emacs, and was just about to switch to vim.
browser opera (desktop & phone) i work at opera for a reason ;-P
music player spotify not much to say, unified streaming choice across devices
terminal GNOME terminal alacritty i'm interested in building my own terminal as well... we'll see if that ever happens.
communication discord (desktop), SMS slack, gmail for work and organizations i am using slack or gmail. since i have a dumb phone i currently only use SMS on my personal phone.

hardware

category home work description
stationary computer custom-built desktop computers i will get rid of my desktop computer at work soon enough, so i won't dig into the specs because of that. i might list my personal desktop computer specs soon
laptop macbook m4 pro i recently switched to a macbook, mainly to run xcode and build for ios. i also have a lenovo thinkpad i use for ethical hacking stuff.
keyboard ducky origin vintage glove80 i use an ergonomic split keyboard at work, since i spend hours writing code there every day.
mouse logitech mx vertical logitech mx ergo i really like the logitech mx series.
phone motorola g06 (with dumb phone launcher) & light phone III n/a i used to own a motorola g05 which i ended up buying for less than $90. however, in my sleep, i accidentally threw this phone in the wall so it broke. so i bought the g06 instead (for less than $90 actually), as it also has NFC functionality. i use this for banking apps, and similar. my primary phone will be the light phone III (as soon as it arrives, probably in january).
speakers mackie cr n/a i bought them back in highschool and they are still getting the job done
microphone shure sm 7 b + behringer xenyx q502 sound interface n/a when i make discord calls, i use this microphone. besides that, i sometimes record screencasts and similar content. every now and then, i also record vocals for songs i produce.
headphones sonos ace i would describe them as a lightweight android alternative to airpods max. i really love them and don't see myself replacing them anytime soon.
camera canon eos 500 (film), nikon coolpix p510 (ultrazoom), nikon D850 (DSLR), IMX789 sensor fine-tuned by hasselblad (casual) n/a for film photography, i'm using a canon eos 500 that i found in a second-hand store. the nikon coolpix p510 used to be my main camera, it has a really nice optical zoom, which you won't find in any smartphone, for example. i like this because you can take good photos from afar, and for people interested in birds, like me, it's a great feature. i just recently bought the nikon d850 which is known for being the best DSLR camera out there. for more casual photos, i use a oneplus phone dedicated solely to photography.
note taking remarkable 2 tablet, paperblanks (physical) for journaling, sketching or quick notes
clock casio f-91w digital wrist clock, it's nice for keeping track of time! although it's currently a few minutes behind, i use that as a feature, haha. i like to be on time.
